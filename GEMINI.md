# Laturaivo Project Structure

This document outlines the file structure for the Laturaivo game.

## Root Directory
- `index.html`: The main entry point of the game.
- `GEMINI.md`: This file.
- `README.md`: Project overview and instructions.

## CSS (`css/`)
- `main.css`: Main stylesheet for the game.

## JavaScript (`js/`)
- `game.js`: Core game logic.
- `engine.js`: General game engine functionalities (rendering, physics, etc.).
- `ui.js`: UI management.
- `sfx.js`: Sound effects and music handling.

## Assets (`assets/`)

### Audio (`assets/audio/`)
- `music1.mp3`: Background music track 1.
- `music2.mp3`: Background music track 2.
- `music3.mp3`: Background music track 3.
- `sfx_pole_strike.mp3`: Sound effect for the pole strike.
- `sfx_slalom_slash.mp3`: Sound effect for the slalom slash.
- `sfx_nordic_boost.mp3`: Sound effect for the Nordic boost.
- `sfx_enemy_hit.mp3`: Sound effect for an enemy being hit.
- `sfx_player_hit.mp3`: Sound effect for the player being hit.
- `sfx_game_over.mp3`: Game over sound effect.

### Backgrounds (`assets/backgrounds/`)
- `forest_day.png`: Daytime forest background.
- `forest_night.png`: Nighttime forest background.

### Sprites (`assets/sprites/`)
- `skier.png`: Player character sprite.
- `hiker.png`: Hiker enemy sprite.
- `rabbit.png`: Rabbit enemy sprite.
- `bird.png`: Bird enemy sprite.
